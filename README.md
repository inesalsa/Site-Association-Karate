# 🥋 Projet Web : Association de Karaté

Projet réalisé dans le cadre du **Semestre 4 de Licence de Mathématiques et Informatique applisuées aux sciences humaines et sociales parcours MIAGE**. Ce site permet de gérer la communication et l'organisation d'un club de karaté.

## 📝 Description
L'objectif est de proposer une plateforme complète pour les pratiquants et les administrateurs du club. Le site permet de consulter les informations essentielles (horaires, cours, tarifs) et de gérer les interactions avec les adhérents.

## ✨ Fonctionnalités
- **Accueil** : Présentation du club et des actualités.
- **Gestion des cours** : Affichage des différents styles et horaires.
- **Espace Membre** : Connexion et inscription pour les adhérents.
- **Back-office** : Interface pour administrer les contenus (ajout/modification de membres ou d'événements).
- **Contact** : Formulaire fonctionnel pour les demandes d'informations.

## 🛠️ Technologies utilisées
- **Frontend** : HTML5, CSS3, JavaScript.
- **Backend** : PHP.
- **Base de données** : MySQL.
- **Design** : Responsive (adapté aux mobiles et tablettes).
